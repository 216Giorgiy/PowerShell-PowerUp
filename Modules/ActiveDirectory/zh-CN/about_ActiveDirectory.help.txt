主题
    Active Directory 模块概述

简短说明
    Active Directory 模块是用于管理 Active Directory 的命令行界面。

详细说明
    Windows PowerShell 的 Active Directory 模块供管理和操作 Active Directory
    的 IT 专业人员使用。Active Directory 模块提供了一种高效的工作方式，用于在
    Active Directory 域服务 (AD DS) 和 Active Directory 轻型目录服务 (AD LDS)
    实例环境中为这些实例完成许多管理、配置和诊断任务。Active Directory 模块包括
    一组 Windows PowerShell cmdlet 和一个提供程序。该提供程序通过分层导航系
    统（非常类似于文件系统）公开 Active Directory 数据库。与文件系统中的驱动器
   （如 C:）一样，可以将 Windows PowerShell 驱动器连接至 Active Directory 域和
    AD LDS 以及 Active Directory 快照。

  Active Directory 模块 Cmdlet 的功能范围
    通过 New-ADUser、Get-ADOrganizationalUnit、Set-ADComputer 和 Remove-ADUser
    等 cmdlet 可以对 Active Directory 对象执行创建、读取、更新和删除操作。

    通过 Enable-ADAccount、Unlock-ADAccount、New-ADServiceAccount、
    Set-ADAccountControl 和 Remove-ADFineGrainedPasswordPolicy 等 cmdlet 可以
    执行帐户和密码策略管理。

    通过 Get-ADForest、Set-ADForest、Set-ADForestMode、Enable-ADOptionalFeature、
    Get-ADDomainController 和 Get-ADDomain 等 cmdlet 可以执行域和林管理。

  列出 Active Directory 模块 Cmdlet
    若要获取所有 Active Directory 模块 cmdlet 的列表，请参阅 get-command -module
    ActiveDirectory。

  开始使用
    只需单击下面的快捷方式，即可方便地开始使用 Windows PowerShell 的 Active Directory
    模块：

       “开始”->“管理工具”->
       适用于 Windows PowerShell 的 Active Directory 模块

    此快捷方式可打开预加载了 Active Directory 模块的 Windows PowerShell。或者，
    可以在任何 Windows PowerShell 提示符下键入下面的命令来导入 Active Directory
    模块：

      import-module ActiveDirectory
