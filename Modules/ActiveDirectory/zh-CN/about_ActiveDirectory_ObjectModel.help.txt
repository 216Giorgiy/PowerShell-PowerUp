主题
    关于 Active Directory 对象模型

简短说明
    说明 Windows PowerShell 的 Active Directory 模块对象模型。

详细说明
    本主题说明用于对实际 Active Directory 属性 (Attribute) 进行
    建模的 Active Directory模块类及其属性 (Property)。此外，还
    概述从 Active Directory 对应项构造的类层次结构。该对象模型
    为 Active Directory 模块 cmdlet 所支持的所有操作建立数据基础。

  类层次结构
    下面的列表演示在 Active Directory 模块对象模型中定义的类层次结构，其中
    以缩进表示类继承。此继承模型允许 Active Directory cmdlet 接受一组对象类
    型作为输入。例如，这表示 cmdlet Get-ADPrincipalGroupMembership 可以接受
    以下任何对象作为输入：ADGroup、ADAccount、ADComputer、ADServiceAccount 
    或 ADUser。可以这样做是因为采用了继承模型，从而保证 ADUser 对象
    具有 ADPrincipal 对象的所有属性。

    ADEntity
      ADRootDSE
      ADObject
        ADFineGrainedPasswordPolicy
        ADOptionalFeature
        ADOrganizationalUnit
        ADPartition
          ADDomain
        ADPrincipal
          ADAccount
            ADComputer
            ADServiceAccount
            ADUser
          ADGroup
      ADDefaultDomainPasswordPolicy
      ADForest
      ADDirectoryServer
        ADDomainController

    Active Directory 模块类
      下面的列表演示类层次结构列表中的每个 Active Directory 模块类。每个类
      都定义了一组属性 (Property)，其中一些是默认情况下检索的 LDAP 
      属性 (Attribute)，还有一些是为 Active Directory 模块专门创建的新
      属性 (Property)。这些新属性 (Property) 派生自一个或多个 LDAP 
      属性 (Attribute)（如类列表中所示）。

    ADEntity
      派生所有其他类的基级类。
