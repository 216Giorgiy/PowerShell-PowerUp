    Active Directory オブジェクト モデルについて

簡易説明
    Windows PowerShell の Active Directory モジュールのオブジェクト モデルにつ
    いて説明します。

詳細説明
    このトピックでは、実際の Active Directory 属性のモデル化に使用される 
    Active Directory モジュールのクラスとそのプロパティについて説明します。また、
    Active Directory のクラス階層から構築されるクラス階層についても説明します。
    オブジェクト モデルは、Active Directory モジュールのコマンドレットでサポート
    されているすべての操作に対するデータの基礎となります。

  クラス階層
    次の一覧に、Active Directory モジュールのオブジェクト モデルで定義されて
    いるクラス階層を示し、クラスの継承をインデントで示します。この継承モデル
    により、Active Directory コマンドレットはさまざまな種類のオブジェクトを
    入力として受け入れることができます。
    たとえば Get-ADPrincipalGroupMembership コマンドレットの場合、ADGroup、
    ADAccount、ADComputer、ADServiceAccount、または ADUser オブジェクトを入力と
    して受け入れることができます。これを可能にしているのがこの継承モデルです。
    これにより、ADUser オブジェクトには ADPrincipal オブジェクトのすべてのプロ
    パティが確実に含まれます。

    Active Directory モジュールのクラス
      次に、クラス階層の一覧で示したすべての Active Directory モジュールのクラス
      について説明します。各クラスは一連のプロパティを定義します。その一部は既定
      で取得される LDAP 属性です。また、Active Directory モジュール用に特に作成
      された新しいプロパティもあります。これらの新しいプロパティは、クラスの
      一覧で示すように、1 つまたは複数の LDAP 属性から派生しています。

    ADEntity
      基礎となるレベルのクラスであり、他のすべてのクラスはこのクラスから派生し
      ます。

    ADRootDSE

      る場合があります。

      から派生する、System.String 型のプロパティ。

      型のプロパティ。

      DefaultNamingContext: ディレクトリ属性 DefaultNamingContext から派生
      する、System.String 型のプロパティ。

      DnsHostName: ディレクトリ属性 DnsHostName から派生する、System.String
      型のプロパティ。

      DomainControllerFunctionality: ディレクトリ属性 
      DomainControllerFunctionality から派生する、ADDomainControllerMode 型の
      プロパティ。

      DomainFunctionality: ディレクトリ属性 DomainFunctionality から派生する、
      ADDomainMode 型のプロパティ。

      DsServiceName: ディレクトリ属性 DsServiceName から派生する、System.String
      型のプロパティ。

      ForestFunctionality: ディレクトリ属性 ForestFunctionality から派生する、

      System.Boolean 型のプロパティ。

      HighestCommittedUSN: ディレクトリ属性 HighestCommittedUSN から派生する、
      System.Long 型のプロパティ。

      LdapServiceName: ディレクトリ属性 LdapServiceName から派生する、
      System.String 型のプロパティ。

      NamingContexts: ディレクトリ属性 NamingContexts から派生する、
      System.String 型のプロパティ。

      RootDomainNamingContext: ディレクトリ属性 RootDomainNamingContext から
      派生する、System.String 型のプロパティ。

      SchemaNamingContext: ディレクトリ属性 SchemaNamingContext から派生する、
      System.String 型のプロパティ。

      ServerName: ディレクトリ属性 ServerName から派生する、System.String 型
      のプロパティ。

      SubschemaSubentry: ディレクトリ属性 SubschemaSubentry から派生する、
      ADObject 型のプロパティ。

      SupportedCapabilities: ディレクトリ属性 SupportedCapabilities から派生

      SupportedControl: ディレクトリ属性 SupportedControl から派生する、
      ADObjectIdentifier 型のプロパティ。
      

      SupportedLDAPPolicies: ディレクトリ属性 SupportedLDAPPolicies から派生
      する、System.String 型のプロパティ。

      から派生する、System.Boolean 型のプロパティ。

      LockoutDuration: ディレクトリ属性 msDS-LockoutDuration から派生する、
      System.TimeSpan 型のプロパティ。

      LockoutObservationWindow: ディレクトリ属性 msDS-LockoutObservationWindow 
      から派生する、System.TimeSpan 型のプロパティ。
      

      LockoutThreshold: ディレクトリ属性 msDS-LockoutThreshold から派生する、
      System.Int32 型のプロパティ。

      MaxPasswordAge: ディレクトリ属性 msDS-MaximumPasswordAge から派生する、
      System.TimeSpan 型のプロパティ。

      MinPasswordAge: ディレクトリ属性 msDS-MinimumPasswordAge から派生する、
      System.TimeSpan 型のプロパティ。

      MinPasswordLength: ディレクトリ属性 msDS-MinimumPasswordLength から派生
      する、System.Int32 型のプロパティ。

      PasswordHistoryCount: ディレクトリ属性 msDS-PasswordHistoryLength から
      派生する、System.Int32 型のプロパティ。

      Precedence: ディレクトリ属性 msDS-PasswordSettingsPrecedence から派生
      する、System.Int32 型のプロパティ。

      ReversibleEncryptionEnabled: ディレクトリ属性 
      msDS-PasswordReversibleEncryptionEnabled から派生する、System.Boolean 型
      のプロパティ。

    ADPrincipal
      ユーザー型、コンピューター型、グループ型、または iNetOrgPerson 型の 
      Active Directory オブジェクトであるセキュリティ プリンシパルを表し、
      ADObject から派生しています。

      ADPrincipal には、親から継承したプロパティに加えて、次のプロパティも
      含まれる場合があります。

      HomePage: ディレクトリ属性 wWWHomePage から派生する、System.String 型の
      プロパティ。

      MemberOf: ディレクトリ属性 memberOf から派生する、System.String 型のプロ
      パティ。

      SamAccountName: ディレクトリ属性 sAMAccountName から派生する、
      System.String 型のプロパティ。このプロパティは AD LDS ではサポートされて
      いません。

      SID: ディレクトリ属性 objectSid から派生する、
      System.Security.Principal.SecurityIdentifier 型のプロパティ。
      

      SIDHistory: ディレクトリ属性 sIDHistory から派生する、
      System.Security.Principal.SecurityIdentifier 型のプロパティ。このプロパ
      ティは AD LDS ではサポートされていません。

    ADAccount
      セキュリティ アカウント (ユーザー型、コンピューター型、または 
      iNetOrgPerson 型の Active Directory オブジェクト) を表し、ADPrincipal 
      から派生しています。

      ADAccount には、親から継承したプロパティに加えて、次のプロパティも
      含まれる場合があります。

      AccountExpirationDate: ディレクトリ属性 accountExpires から派生する、
      System.DateTime 型のプロパティ。

      AccountLockoutTime: ディレクトリ属性 lockoutTime から派生する、
      System.DateTime 型のプロパティ。

      AccountNotDelegated: ディレクトリ属性 userAccountControl および 
      msDS-User-Account-Control-Computed から派生する、System.Boolean 型のプロ
      パティ。このプロパティは AD LDS ではサポートされていません。

      AllowReversiblePasswordEncryption: System.Boolean 型のプロパティ。
      派生元のディレクトリ属性は、AD DS の場合は userAccountControl、
      AD LDS の場合は ms-DS-UserEncryptedTextPasswordAllowed です。

      BadLogonCount: ディレクトリ属性 badPwdCount から派生する、System.Int32
      型のプロパティ。

      CannotChangePassword: ディレクトリ属性 nTSecurityDescriptor から派生す
      る、System.Boolean 型のプロパティ。

      Certificates: ディレクトリ属性 userCertificate から派生する、
      System.Security.Cryptography.X509Certificates.X509Certificate 型のプロパ
      ティ。

      DoesNotRequirePreAuth: ディレクトリ属性 userAccountControl および 
      msDS-User-Account-Control-Computed から派生する、System.Boolean 型の
      プロパティ。このプロパティは AD LDS ではサポートされていません。

      Enabled: System.Boolean 型のプロパティ。派生元のディレクトリ属性は、
      AD DS の場合は userAccountControl および 
      msDS-User-Account-Control-Computed、AD LDS の場合は 
      msDS-UserAccountDisabled です。

      HomedirRequired: ディレクトリ属性 userAccountControl および 
      msDS-User-Account-Control-Computed から派生する、System.Boolean 型の
      プロパティ。このプロパティは AD LDS ではサポートされていません。

      LastBadPasswordAttempt: ディレクトリ属性 badPasswordTime から派生する、
      System.DateTime 型のプロパティ。

      LastLogonDate: ディレクトリ属性 lastLogonTimestamp から派生する、
      System.DateTime 型のプロパティ。

      LockedOut: System.Boolean 型のプロパティ。派生元のディレクトリ属性は、
      AD DS の場合は userAccountControl、msDS-User-Account-Control-Computed
      および lockoutTime で、AD LDS の場合は msDS-UserAccountDisabled です。

      MNSLogonAccount: ディレクトリ属性 userAccountControl および 
      msDS-User-Account-Control-Computed から派生する、System.Boolean 型の
      プロパティ。このプロパティは AD LDS ではサポートされていません。

      PasswordExpired: System.Boolean 型のプロパティ。派生元のディレクトリ属性
      は、AD DS の場合は userAccountControl および 
      msDS-User-Account-Control-Computed、AD LDS の場合は 
      msDS-UserPasswordExpired です。
      

      PasswordLastSet: ディレクトリ属性 pwdLastSet から派生する、
      System.DateTime 型のプロパティ。

      PasswordNeverExpires: System.Boolean 型のプロパティ。派生元のディレクト
      リ属性は、AD DS の場合は userAccountControl および 
      msDS-User-Account-Control-Computed、AD LDS の場合は 
      msDS-UserDontExpirePassword です。

      PasswordNotRequired: System.Boolean 型のプロパティ。派生元のディレクトリ
      属性は、AD DS の場合は userAccountControl および 
      msDS-User-Account-Control-Computed、AD LDS の場合は 
      ms-DS-UserPasswordNotRequired です。

      PrimaryGroup: ディレクトリ属性 primaryGroupID および objectSid から派生
      する、System.String 型のプロパティ。このプロパティは AD LDS ではサポート
      されていません。

      ServicePrincipalNames: ディレクトリ属性 servicePrincipalName から派生
      する、System.String 型のプロパティ。このプロパティは AD LDS ではサポート

      TrustedForDelegation: ディレクトリ属性 userAccountControl および

      TrustedToAuthForDelegation: ディレクトリ属性 userAccountControl および
      msDS-User-Account-Control-Computed から派生する、System.Boolean
      型のプロパティ。このプロパティは AD LDS ではサポートされていません。

      UseDESKeyOnly: ディレクトリ属性 userAccountControl および 
      msDS-User-Account-Control-Computed から派生する、System.Boolean 型のプロ

      UserPrincipalName: ディレクトリ属性 userPrincipalName から派生する、
