    Active Directory モジュールの概要

簡易説明
    Active Directory モジュールは Active Directory を管理するためのコマンド ラ
    イン インターフェイスです。

詳細説明
    Windows PowerShell の Active Directory モジュールは、Active Directory
    の管理と運用を行う IT プロフェッショナルのために用意されたモジュールです。
    Active Directory モジュールを使用すると、それぞれの環境で、Active Directory 
    ドメイン サービス (AD DS) インスタンスおよび Active Directory ライトウェ
    イト ディレクトリ サービス (AD LDS) インスタンスにおける多数の管理タスク、
    構成タスク、および診断タスクを効率的に実行できます。Active Directory 
    モジュールには、一連の Windows PowerShell コマンドレットと 1 つのプロバイ
    ダーが含まれています。このプロバイダーは、ファイル システムによく似た階層的
    なナビゲーション システムを介して Active Directory データベースを公開しま
    す。ファイル システム内のドライブ (C: など) と同様に、Windows PowerShell 
    ドライブを Active Directory ドメイン、AD LDS、および Active Directory 
    スナップショットに接続できます。

  Active Directory モジュールのコマンドレットの適用範囲
    Active Direcotry オブジェクトに対する Create、Read、Update、および Delete
    アクションは、New-ADUser、Get-ADOrganizationalUnit、Set-ADComputer、
    Remove-ADUser などのコマンドレットでサポートされています。

    アカウント管理とパスワード ポリシー管理は、Enable-ADAccount、
    Unlock-ADAccount、New-ADServiceAccount、Set-ADAccountControl、
    Remove-ADFineGrainedPasswordPolicy などのコマンドレットでサポートされてい
    ます。
    

    ドメイン管理とフォレスト管理は、Get-ADForest、Set-ADForest、
    Set-ADForestMode、Enable-ADOptionalFeature、Get-ADDomainController、
    Get-ADDomain などのコマンドレットでサポートされています。

  Active Directory モジュールのコマンドレットの一覧表示
    Active Directory モジュールのすべてのコマンドレットの一覧を取得するには、
    「get-command-module ActiveDirectory」を参照してください。

  はじめに
    Windows PowerShell の Active Directory モジュールでの作業は、次のショート
    カットをクリックして簡単に開始できます。

       [スタート] ボタン → [管理ツール] →
       [Windows PowerShell 用の Active Directory モジュール]

    このショートカットをクリックすると、Active Directory モジュールがプリロー
    ドされた状態で Windows PowerShell が開きます。また、Windows PowerShell 
    プロンプトで次のコマンドを入力して、Active Directory モジュールをインポー
    トすることもできます。

      import-module ActiveDirectory
