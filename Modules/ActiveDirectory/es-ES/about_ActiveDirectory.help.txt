TEMA
    Información general del módulo de Active Directory

DESCRIPCIÓN BREVE
    El módulo de Active Directory es una interfaz de línea de comandos
    para administrar Active Directory.

DESCRIPCIÓN DETALLADA
    El módulo de Active Directory para Windows PowerShell está
    dirigido a los profesionales de tecnologías de la información
    que administran e interactúan con Active Directory. El módulo
    de Active Directory proporciona un medio eficaz para llevar a
    cabo numerosas tareas administrativas, de configuración y de
    diagnóstico en instancias de Servicios de dominio de Active
    Directory (AD DS) y de Active Directory Lightweight Directory
    Services (DC LDS) en sus entornos. El módulo de Active Directory
    incluye un conjunto de cmdlets de Windows PowerShell y un
    proveedor. El proveedor expone la base de datos de Active
    Directory a través de un sistema de navegación jerárquico muy
    similar al sistema de archivos. Al igual que sucede con las
    unidades de disco de un sistema de archivos, como C:, puede
    conectar las unidades de Windows PowerShell a los dominios de
    Active Directory y AD LDS, así como a las instantáneas de Active
    Directory.

  Cobertura de cmdlets del módulo de Active Directory
    Las acciones de crear, leer, actualizar y eliminar en objetos de
    Active Directory se realizan mediante cmdlets como New-ADUser,
    Get-ADOrganizationalUnit, Set-ADComputer y Remove-ADUser.

    La administración de cuentas y de la Directiva de contraseñas se
    realiza mediante cmdlets como Enable-ADAccount, Unlock
    -ADAccount, New-ADServiceAccount, Set-ADAccountControl y 
    Remove-ADFineGrainedPasswordPolicy.
    

    La administración de dominios y de bosques se lleva a cabo
    mediante cmdlets como Get-ADForest, Set-ADForest, 
    Set-ADForestMode, Enable-ADOptionalFeature, Get-ADDomainController y 
    Get-ADDomain.
    

  Mostrar una lista de los cmdlets del módulo de Active Directory
    Para obtener una lista de todos los cmdlets del módulo de Active
    Directory, vea get-command -module ActiveDirectory.

  Introducción
    Empezar a trabajar con el módulo de Active Directory para Windows
    PowerShell es tan fácil como hacer clic en el acceso
    directo siguiente:

       Inicio -> Herramientas administrativas ->
       Módulo de Active Directory para Windows PowerShell

    Este acceso directo abre Windows PowerShell con el módulo de
    Active Directory cargado. Además, puede escribir el comando
    siguiente en un símbolo del sistema de Windows PowerShell para
    importar el módulo de Active Directory:

      import-module ActiveDirectory
