主題
    Active Directory 模組概觀

簡短描述
    Active Directory 模組是用於管理 Active Directory 的命令列介面。

完整描述
    Windows PowerShell 的 Active Directory 模組適用於負責管理和操作
    Active Directory 的 IT 專業人員，Active Directory 模組可提供有效的方法，讓
    IT 專業人員在其環境下完成 Active Directory 網域服務 (AD DS) 與
    Active Directory 輕量型目錄服務 (AD LDS) 執行個體的許多系統管理、設定和診
    斷工作。Active Directory 模組包含一組 Windows PowerShell Cmdlet 與一個提供
    者。此提供者會透過與檔案系統類似的階層式瀏覽系統來公開 Active Directory 資
    料庫。和檔案系統的磁碟機 (例如 C:) 一樣，您可以將 Windows PowerShell 磁碟
    機連線到 Active Directory 網域與 AD LDS，以及 Active Directory 快照。

  Active Directory 模組 Cmdlet 的範圍
    New-ADUser、Get-ADOrganizationalUnit、Set-ADComputer 和 Remove-ADUser
    這類 Cmdlet 都支援建立、讀取、更新和刪除 Active Directory 物件。

    Enable-ADAccount、Unlock-ADAccount、New-ADServiceAccount、
    Set-ADAccountControl 和 Remove-ADFineGrainedPasswordPolicy 這類 Cmdlet
    支援帳戶及密碼原則管理。

    Get-ADForest、Set-ADForest、Set-ADForestMode、Enable-ADOptionalFeature、
    Get-ADDomainController 和 Get-ADDomain 這類 Cmdlet 支援網域及樹系管理。

  列出 Active Directory 模組 Cmdlet
    若想取得所有 Active Directory 模組 Cmdlet 的清單，請參閱
    get-command -module ActiveDirectory。

  開始使用
    只要按一下下列捷徑，便可輕鬆開始使用 Windows PowerShell 的 Active Directory
    模組：

       [開始] -> [系統管理工具] ->
       Windows PowerShell 的 Active Directory 模組

    這個捷徑隨即以預先載入的 Active Directory 模組開啟 Windows PowerShell。此
    外，您還可以在任何 Windows PowerShell 提示中輸入下列命令，即可匯入 Active
    Directory 模組：

      import-module ActiveDirectory

    以下前兩個主題提供 Active Directory 模組和 Active Directory 提供者的高階概
    觀。

    如需 Windows PowerShell 之 Active Directory 提供者的簡短介紹，請參閱
    ActiveDirectory。

    下列主題是 Active Directory 模組 Cmdlet 的概念性支援主題。

    如需 -Identity 參數的介紹 (Active Directory 模組 Cmdlet 使用此參數來識別目
    錄中的物件)，請參閱 about_ActiveDirectory_Identity。

    如需 -Filter 參數的介紹 (Active Directory 模組 Cmdlet 使用此參數來搜尋目錄
    中的物件)，請參閱 about_ActiveDirectory_Filter。

    如需 Active Directory 模組實作之 .NET Framework 架構物件模型的介紹，請參閱
    about_ActiveDirectory_ObjectModel。
