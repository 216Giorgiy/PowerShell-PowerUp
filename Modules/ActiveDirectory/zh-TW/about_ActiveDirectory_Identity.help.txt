主題
    Active Directory 物件識別

簡短描述
    Windows PowerShell 物件的 Active Directory 模組有一系列的識別屬性，可以用來
    進行搜尋及擷取。

完整描述
    為了識別 Active Directory 的物件，每個物件都有可做為識別元的屬性。在
    Active Directory 模組中，Identity 參數可用來傳遞物件的識別值。每個物件型別
    都有自己一組的可能型別和值，供 Identity 參數使用。請參閱所指定 Cmdlet
    之 Identity 參數的詳細說明，以取得其使用方式的詳細資訊。
    
    以 Active Directory 模組 Cmdlet 進行搜尋時，Identity 參數的值與 Server 和
    Partition 參數的值是一起用來唯一識別單一物件。Server 參數是用來尋找要連線到
    的目標伺服器。Partition 參數會進一步將搜尋目標縮小為特定的磁碟分割，然後
    Identity 參數會解析成磁碟分割中的唯一物件。
    
    如果使用識別解析找到一個以上的物件，Active Directory 模組便會擲回錯誤。

    如需 Server 和 Partition 參數的詳細資訊，請參閱使用這些參數之個別 Cmdlet
    (如 Get-ADUser) 的說明主題，請輸入：


        get-help Get-ADUser


  物件與識別元
    每個物件都有可做為物件識別的屬性清單。此外，如果物件繼承自另一個物件，則父
    物件的識別也可做為子物件的識別。如需 Active Directory 物件階層的詳細資訊，
    請參閱 about_ActiveDirectory_ObjectModel。

    注意：如果是 Active Directory 提供者 Cmdlet，則只有物件的「分辨名稱」或「相
    對分辨名稱」可做為識別。如需 Active Directory 提供者 Cmdlet 的清單，
    請參閱 ActiveDirectory。

  識別屬性
    下面依照物件型別列出識別屬性。

    ADAccount
      分辨名稱
      GUID (objectGUID)
      安全性識別元 (objectSid)
      SAM 帳戶名稱 (sAMAccountName)
      
    ADComputer
      分辨名稱
      GUID (objectGUID)
      安全性識別元 (objectSid)
      安全性帳戶管理員帳戶名稱 (sAMAccountName)

    ADDirectoryServer
      伺服器物件的名稱 (name)
        若是 AD LDS 執行個體，則名稱的語法為
          <電腦名稱>$<執行個體名稱>
        若是其他 Active Directory 執行個體，則使用名稱屬性的值。
        NTDS 設定物件的分辨名稱
        代表目錄伺服器之伺服器物件的分辨名稱
        設定分割底下伺服器物件的 GUID (objectGUID)
        設定分割底下 NTDS 設定物件的 GUID (objectGUID)

    ADDomain
      分辨名稱
      GUID
      安全性識別元
      DNS 網域名稱
      NetBIOS 網域名稱

    ADDomainController
      GUID (objectGUID)
      IPV4 位址
      全域 IPV6 位址
      DNS 主機名稱 (dNSHostName)
      伺服器物件的名稱
      NTDS 設定物件的分辨名稱
      代表網域控制站之伺服器物件的分辨名稱
      設定分割底下 NTDS 設定物件的 GUID
      設定分割底下伺服器物件的 GUID 
      代表網域控制站之電腦物件的分辨名稱

    ADFineGrainedPasswordPolicy
      分辨名稱
      GUID (objectGUID)
      名稱 (name)
      
    ADForest
      GUID (objectGUID)
      DNS 主機名稱
      NetBIOS 名稱

    ADGroup
      分辨名稱
      GUID (objectGUID)
      安全性識別元 (objectSid)
      安全性帳戶管理員 (SAM) 帳戶名稱 (sAMAccountName)

    ADObject
      分辨名稱
      GUID (objectGUID)

    ADOptionalFeature
      分辨名稱
      名稱 (name)
      功能 GUID (featureGUID)
      GUID (objectGUID)

    ADOrganizationalUnit
      分辨名稱
      GUID (objectGUID)

    ADPrincipal
      分辨名稱
      GUID (objectGUID)
      安全性識別元 (objectSid)
      SAM 帳戶名稱 (sAMAccountName)

    ADServiceAccount
      分辨名稱
      GUID (objectGUID)
      安全性識別元 (objectSid)
      SAM 帳戶名稱 (sAMAccountName)

    ADUser
      分辨名稱
      GUID (objectGUID)
      安全性識別元 (objectSid)
      SAM 使用者名稱 (sAMUserName)


  識別格式

    Active Directory 模組物件有一系列的識別屬性。下面列出這些識別屬性、其型別和
    格式。

    分辨名稱
      範例：CN=SaraDavis,CN=Europe,CN=Users,DC=corp,DC=contoso,DC=com

    DNS 網域名稱
      範例：redmond.corp.contoso.com

    DNS 主機名稱 (dNSHostName)
      範例：corp-DC01.corp.contoso.com

    功能 GUID (featureGUID)
      範例：599c3d2e-f72d-4d20-8a88-030d99495f20

    完整的網域名稱
      範例：corp.contoso.com

    全域 IPV6 位址
      範例：2001:4898:0:fff:200:5efe:157.59.132.61

    GUID (objectGUID)
      範例：599c3d2e-f72d-4d20-8a88-030d99495f20

    IPV4 位址
      範例：157.59.132.61

    NetBIOS 網域名稱
      範例：redmond

    伺服器物件的名稱
      範例：corp-DC01$

    安全性帳戶管理員 (SAM) 帳戶名稱 (sAMAccountName)    
      範例：saradavisreports

    安全性識別元 (objectSid)
      範例：S-1-5-21-3165297888-301567370-576410423-1103

    名稱
      範例：Recycle Bin Feature
