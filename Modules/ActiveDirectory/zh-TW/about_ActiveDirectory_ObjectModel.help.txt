主題
    關於 Active Directory 物件模型

簡短描述
    描述 Windows PowerShell 之 Active Directory 模組的物件模型。

完整描述
    本主題說明用來塑模實際 Active Directory 屬性 (Attribute) 之 
    Active Directory 模組類別及其屬性 (Property)，同時也概述從 
    Active Directory 對應項建構的類別階層。物件模型會為 
    Active Directory 模組 Cmdlet 所支援的所有作業建立資料基礎。

  類別階層
    下列清單以縮排方式顯示 Active Directory 模組物件模型中定義的類別階層。此繼
    承模型允許 Active Directory Cmdlet 接受某範圍的物件型別做為輸入。舉例來說，
    這表示 Cmdlet Get-ADPrincipalGroupMembership 可以接受輸入下列任一物件：
    ADGroup、ADAccount、ADComputer、ADServiceAccount 或 ADUser。由於是繼承模型
    ，可確保 ADUser 物件具有 ADPrincipal 物件的所有屬性，因此以上方式可行。

    ADEntity
      ADRootDSE
      ADObject
        ADFineGrainedPasswordPolicy
        ADOptionalFeature
        ADOrganizationalUnit
        ADPartition
          ADDomain
        ADPrincipal
          ADAccount
            ADComputer
            ADServiceAccount
            ADUser
          ADGroup
      ADDefaultDomainPasswordPolicy
      ADForest
      ADDirectoryServer
        ADDomainController

    Active Directory 模組類別
      下列清單顯示類別階層清單中的每個 Active Directory 模組類別。每個類別都定
      義一組屬性 (Property)，其中有些是預設會擷取的 LDAP 屬性 (Attribute)，有
      些則是專為 Active Directory 模組建立的新屬性 (Property)。這些新屬性
      (Property) 衍生自一個或多個 LDAP 屬性 (Attribute)，如類別清單所述。

      衍生自所有其他類別的基底層級類別。
