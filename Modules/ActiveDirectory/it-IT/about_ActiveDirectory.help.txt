ARGOMENTO
    Cenni preliminari sul modulo di Active Directory

DESCRIZIONE BREVE
    Il modulo di Active Directory è un'interfaccia della riga di
    comando per la gestione di Active Directory.

DESCRIZIONE DETTAGLIATA
    Il modulo di Active Directory per Windows PowerShell è destinato a
    professionisti IT che gestiscono e utilizzano Active Directory.
    Il modulo di Active Directory consente di eseguire in modo
    efficace numerose attività di amministrazione, configurazione
    e diagnostica per le istanze di Servizi di dominio Active
    Directory e Active Directory Lightweight Directory Services
    (AD LDS) nei relativi ambienti. Il modulo di Active Directory
    include un insieme di cmdlet di Windows PowerShell e un
    provider. Il provider espone il database di Active Directory
    attraverso un sistema di esplorazione gerarchico molto simile
    al file system. Come con le unità di un file system, ad esempio
    C, è possibile connettere le unità di Windows PowerShell a domini di
    Active Directory e AD LDS, nonché agli snapshot di Active Directory.


  Descrizione dei cmdlet del modulo di Active Directory
    Le azioni di creazione, lettura, aggiornamento ed eliminazione
    sono supportate per gli oggetti Active Directory da cmdlet quali
    New-ADUser, Get-ADOrganizationalUnit, Set-ADComputer e Remove-ADUser.

    La gestione degli account e dei criteri password è supportata da
    cmdlet quali Enable-ADAccount, Unlock-ADAccount, New-ADServiceAccount,
    Set-ADAccountControl e Remove-ADFineGrainedPasswordPolicy.


    La gestione dei domini e delle foreste è supportata da cmdlet quali
    Get-ADForest, Set-ADForest, Set-ADForestMode, Enable-ADOptionalFeature,
    Get-ADDomainController e Get-ADDomain.


  Elenco dei cmdlet del modulo di Active Directory
    Per ottenere un elenco di tutti i cmdlet del modulo di Active
    Directory, vedere get-command -module ActiveDirectory.


  Introduzione
    Per imparare a utilizzare il modulo di Active Directory per
    Windows PowerShell è sufficiente fare clic sul collegamento seguente:

       Start -> Strumenti di amministrazione ->
       Modulo di Active Directory per Windows PowerShell

    Questo collegamento consente di aprire Windows PowerShell con il
    modulo di Active Directory precaricato. Inoltre, è possibile
    digitare il comando seguente in qualsiasi prompt di Windows
    PowerShell per importare il modulo di Active Directory:

      import-module ActiveDirectory
