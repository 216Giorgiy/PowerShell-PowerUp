RUBRIQUE
    Vue d'ensemble du module Active Directory

DESCRIPTION COURTE
    Le module Active Directory est une interface de ligne de commande
    pour la gestion d'Active Directory.

DESCRIPTION LONGUE
    Le module Active Directory pour Windows PowerShell s'adresse aux
    professionnels de l'informatique qui administrent Active
    Directory et interagissent avec celui-ci. Il offre un moyen
    efficace d'exécuter de nombreuses tâches d'administration, de
    configuration et de diagnostic pour les instances des services
    AD DS (Active Directory Domain Services) et AD LDS (Active
    Directory Lightweight Directory Services) dans leur
    environnement. Le module Active Directory inclut un jeu
    d'applets de commande Windows PowerShell et un fournisseur. Le
    fournisseur expose la base de données Active Directory par le
    biais d'un système de navigation hiérarchique, très similaire
    au système de fichiers. Vous pouvez connecter les lecteurs
    Windows PowerShell aux domaines Active Directory, aux services
    AD LDS et aux captures instantanées Active Directory, tout comme
    les lecteurs d'un système de fichiers, par exemple C:.

  Couverture des applets de commande du module Active Directory
    Les opérations de création, lecture, mise à jour et suppression
    sont prises en charge pour les objets Active Directory par des
    applets de commande, telles que New-ADUser, Get-ADOrganizationalUnit,
    Set-ADComputer et Remove-ADUser.

    La gestion des comptes et des stratégies de mot de passe est prise
    en charge par des applets de commande comme Enable-ADAccount,
    Unlock-ADAccount, New-ADServiceAccount, Set-ADAccountControl et
    Remove-ADFineGrainedPasswordPolicy.

    La gestion des domaines et des forêts est prise en charge par des
    applets de commande comme Get-ADForest, Set-ADForest, Set-ADForestMode,
    Enable-ADOptionalFeature, Get-ADDomainController et Get-ADDomain.
    

  Liste des applets de commande du module Active Directory
    Pour obtenir la liste de toutes les applets de commande du module
    Active Directory, consultez get-command -module ActiveDirectory.
    

  Mise en route
    Pour démarrer le module Active Directory pour Windows PowerShell,
    il suffit de cliquer sur le raccourci suivant :

       Démarrer -> Outils d'administration ->
       Module Active Directory pour Windows PowerShell

    Ce raccourci ouvre Windows PowerShell avec le module Active
    Directory préchargé. Par ailleurs, vous pouvez taper la commande
    suivante dans toute invite Windows PowerShell pour importer le
    module Active Directory :

      import-module ActiveDirectory
