항목
    Active Directory 개체 모델 정보

간단한 설명
    Windows PowerShell용 Active Directory 모듈의 개체 모델에 대해 설명합니다.

자세한 설명
    이 항목에서는 실제 Active Directory 특성을 모델링하는 데 사용되는 
    Active Directory 모듈 클래스와 해당 속성에 대해 설명합니다. 
    또한 해당 Active Directory 계층 구조에서 생성된 클래스 계층 구조에 
    대해서도 대략적으로 설명합니다. 개체 모델은 Active Directory 모듈 
    cmdlet에서 지원하는 모든 작업에 대한 데이터 기반을 마련합니다.

  클래스 계층 구조
    다음 목록에서는 Active Directory 모듈 개체 모델에 정의된 클래스 계층 
    구조를 보여 줍니다. 클래스 상속은 들여쓰기로 표시됩니다. 이 상속 모델을 
    통해 Active Directory cmdlet이 일련의 개체 유형을 입력으로 사용할 수 
    있습니다. 즉, 예를 들어 Get-ADPrincipalGroupMembership cmdlet은 ADGroup, 
    ADAccount, ADComputer, ADServiceAccount 또는 ADUser 개체 중 하나를 
    입력으로 사용할 수 있습니다. 이것은 상속 모델 때문에 가능하며 이를 
    통해 ADUser 개체가 ADPrincipal 개체의 모든 속성을 갖게 됩니다.

    Active Directory 모듈 클래스
      다음 목록에서는 클래스 계층 구조 목록에서 모든 Active Directory 
      모듈 클래스를 보여 줍니다. 각 클래스는 속성 집합을 정의하며 이 속성 
      집합에는 기본적으로 검색되는 LDAP 특성과 Active Directory 모듈용으로 
      특정하게 만들어진 새 속성이 포함됩니다. 이러한 새 속성은 클래스 
      목록에 간략하게 설명된 대로 하나 이상의 LDAP 특성에서 파생됩니다.

      LockoutObservationWindow - 디렉터리 특성 msDS-LockoutObservationWindow에서
      파생된 System.TimeSpan 유형의 속성

      LockoutThreshold - 디렉터리 특성 msDS-LockoutThreshold에서 파생된 
      System.Int32 유형의 속성

      MaxPasswordAge - 디렉터리 특성 msDS-MaximumPasswordAge에서 파생된 
      System.TimeSpan 유형의 속성

      MinPasswordAge - 디렉터리 특성 msDS-MinimumPasswordAge에서 파생된 
      System.TimeSpan 유형의 속성

      MinPasswordLength - 디렉터리 특성 msDS-MinimumPasswordLength에서 파
      생된 System.Int32 유형의 속성

      PasswordHistoryCount - 디렉터리 특성 msDS-PasswordHistoryLength에서
      파생된 System.Int32 유형의 속성

      Precedence - 디렉터리 특성 msDS-PasswordSettingsPrecedence에서 파생된
      System.Int32 유형의 속성

      ReversibleEncryptionEnabled - 디렉터리 특성 
      msDS-PasswordReversibleEncryptionEnabled에서 파생된 System.Boolean 
      유형의 속성

    ADPrincipal

    ADAccount
      보안 계정(사용자, 컴퓨터 또는 iNetOrgPerson 유형의 Active Directory 
      개체)을 나타내며 ADPrincipal에서 파생됩니다.

      ADAccount에는 상위 클래스에서 상속한 속성 외에도 다음 속성이 포함될
      수 있습니다.

      AccountExpirationDate - 디렉터리 특성 accountExpires에서 파생된 
      System.DateTime 유형의 속성

      AccountLockoutTime - 디렉터리 특성 lockoutTime에서 파생된 
      System.DateTime 유형의 속성

      AccountNotDelegated - 디렉터리 특성 userAccountControl, 
      msDS-User-Account-Control-Computed에서 파생된 System.Boolean 
      유형의 속성. 이 속성은 AD LDS에서 지원되지 않습니다.

      AllowReversiblePasswordEncryption - AD DS의 경우 디렉터리 특성
      userAccountControl에서 파생되고 AD LDS의 경우 디렉터리 특성

      System.Boolean 유형의 속성

      파생된 System.Boolean 유형의 속성
