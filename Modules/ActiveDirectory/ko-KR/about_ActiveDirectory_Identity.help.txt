항목
    Active Directory 개체 ID

간단한 설명
    Windows PowerShell용 Active Directory 모듈 개체에는 검색에 사용되는 다양한 식별 특성이 있습니다. 

자세한 설명
    개체마다 Active Directory에서 개체를 식별하는 데 사용할 수 있는 특성이 있습니다. Active Directory 
    모듈에서는 Identity 매개 변수를 사용하여 개체의 ID 값을 전달할 수 있으며, 개체 유형마다 Identity 매개 
    변수에 사용할 수 있는 고유의 유형 및 값 집합이 있습니다. 매개 변수 사용에 대한 자세한 내용은 지정된 
    cmdlet의 Identity 매개 변수에 대한 자세한 설명을 참조하십시오.

    Active Directory 모듈 cmdlet을 사용하여 검색할 때 Server 및 Partition 매개 변수 
    값과 함께 Identity 매개 변수 값이 단일 개체를 고유하게 식별하는 데 사용됩니다. Server 매개 변수는 
    연결할 서버를 찾는 데 사용되며, Partition 매개 변수는 검색 범위를 특정 파티션으로 한정합니다. 그러면 
    Identity 매개 변수가 해당 파티션의 고유 단일 개체로 확인됩니다. 

    ID 확인을 사용하여 개체를 두 개 이상 찾은 경우 Active Directory 모듈에서 오류를 throw합니다.

    Server 및 Partition 매개 변수에 대한 자세한 내용은 다음을 입력하여 이들 매개 변수가 사용되는
    개별 cmdlet(예: Get-ADUser)에 대한 도움말 항목을 참조하십시오.

	get-help Get-ADUser


  개체 및 ID
    개체마다 ID로 사용할 수 있는 특성 목록이 있습니다. 또한 개체가 다른 개체에서 상속될 경우 상위 개체의 
    ID를 하위 개체의 ID로 사용할 수 있습니다. Active Directory 개체 계층 구조에 대한 자세한 내용은 
    about_ActiveDirectory_ObjectModel을 참조하십시오.

    참고: Active Directory Provider cmdlet의 경우 개체의 '고유 이름' 또는 '상대 고유 
    이름'만 ID로 사용할 수 있습니다. Active Directory 공급자 cmdlet 목록은 
    ActiveDirectory를 참조하십시오.

  ID 특성
    다음은 개체 유형의 ID 특성 목록입니다.

    ADAccount
      고유 이름 
      GUID(objectGUID) 
      보안 식별자(objectSid) 
      SAM 계정 이름(sAMAccountName) 
      
    ADComputer
      고유 이름 
      GUID(objectGUID) 
      보안 식별자(objectSid) 
      보안 계정 관리자 계정 이름(sAMAccountName) 

    ADDirectoryServer
      서버 개체의 이름(name) 
        AD LDS 인스턴스의 경우 이름 구문은 다음과 같습니다. 
   	  <computer-name>$<instance-name>
        다른 Active Directory 인스턴스의 경우 이름 속성 값을 사용합니다.
        NTDS 설정 개체에 대한 고유 이름
        디렉터리 서버를 나타내는 서버 개체에 대한 고유 이름
        구성 파티션 아래의 서버 개체에 대한 GUID(objectGUID)
        구성 파티션 아래의 NTDS 설정 개체에 대한 GUID(objectGUID)

    ADDomain 
      고유 이름
      GUID
      보안 식별자
      DNS 도메인 이름 
      NetBIOS 도메인 이름 

    ADDomainController
      GUID(objectGUID)
      IPV4Address
      전역 IPV6Address 
      DNS 호스트 이름(dNSHostName)
      서버 개체의 이름
      NTDS 설정 개체의 고유 이름. 
      도메인 컨트롤러를 나타내는 서버 개체의 고유 이름
      구성 파티션에 있는 NTDS 설정 개체의 GUID
      구성 파티션에 있는 서버 개체의 GUID. 
      도메인 컨트롤러를 나타내는 컴퓨터 개체의 DN(고유 이름)입니다.

    ADFineGrainedPasswordPolicy
      고유 이름 
      GUID(objectGUID) 
      이름(name) 
      
    ADForest
      정규화된 도메인 이름
      DNS 호스트 이름
      NetBIOS 이름

    ADGroup
      고유 이름 
      GUID(objectGUID) 
      보안 식별자(objectSid) 
      SAM(보안 계정 관리자) 계정 이름(sAMAccountName) 

    ADObject
      고유 이름 
      GUID(objectGUID) 

    ADOptionalFeature
            고유 이름
      이름(name)
      기능 GUID(featureGUID)

    ADOrganizationalUnit
      고유 이름

    ADPrincipal
      고유 이름 
      GUID(objectGUID)

      SAM 계정 이름(sAMAccountName) 

    ADServiceAccount
      고유 이름

  ID 형식
    Active Directory 모듈 개체에는 다양한 ID 특성이 있습니다. 다음은 이러한 특성과 특성 유형 및 형식의 
    목록입니다.

    고유 이름 
      예: CN=SaraDavis,CN=Europe,CN=Users,DC=corp,DC=contoso,DC=com

    DNS 도메인 이름 
      예: redmond.corp.contoso.com

    DNS 호스트 이름(dNSHostName)
      예: corp-DC01.corp.contoso.com

    기능 GUID(featureGUID) 
      예: 599c3d2e-f72d-4d20-8a88-030d99495f20 

    정규화된 도메인 이름
      예: corp.contoso.com

    전역 IPV6Address 
      예: 2001:4898:0:fff:200:5efe:157.59.132.61

    GUID(objectGUID) 
      예: 599c3d2e-f72d-4d20-8a88-030d99495f20 

    IPV4Address
      예:157.59.132.61

    NetBIOS 도메인 이름 
      예: redmond

    서버 개체의 이름
      예: corp-DC01$

    SAM(보안 계정 관리자) 계정 이름(sAMAccountName) 예: saradavisreports

    보안 식별자(objectSid) 
      예: S-1-5-21-3165297888-301567370-576410423-1103

    이름
      예: Recycle Bin Feature
