    Active Directory 모듈 개요

간단한 설명
    Active Directory 모듈은 Active Directory를 관리하기 위한 명령줄 인터페이스
    입니다.

자세한 설명
    Windows PowerShell용 Active Directory 모듈은 Active Directory를 관리
    하고 Active Directory와 인터페이스하는 IT 전문가를 대상으로 합니다. IT
    전문가는 Active Directory 모듈을 사용하여 자신의 환경에서 AD DS(Active
    Directory Domain Services) 및 AD LDS(Active Directory Lightweight
    Directory Services) 인스턴스에 대한 다양한 관리, 구성 및 진단 작업을 효
    율적으로 수행할 수 있습니다. Active Directory 모듈에는 Windows PowerShell
    cmdlet 집합과 공급자가 포함됩니다. 공급자는 파일 시스템과 매우 비슷한 계층적
    탐색 시스템을 통해 Active Directory 데이터베이스를 표시합니다. 파일 시스템의
    드라이브(예: C:)와 마찬가지로 Windows PowerShell 드라이브를 Active Directory
    도메인, AD LDS 및 Active Directory 스냅숏에 연결할 수 있습니다.


  Active Directory 모듈 Cmdlet의 적용 범위
    New-ADUser, Get-ADOrganizationalUnit, Set-ADComputer 및 Remove-
    ADUser cmdlet을 통해 Active Directory 개체에 대한 만들기, 읽기, 업데이트 및
    삭제 작업이 지원됩니다.

    Enable-ADAccount, Unlock-ADAccount, New-ADServiceAccount, Set-ADAccountControl
    및 Remove-ADFineGrainedPasswordPolicy cmdlet을 통해 계정 및 암호 정책 관리가
    지원됩니다.

    Get-ADForest, Set-ADForest, Set-ADForestMode, Enable-ADOptionalFeature, 
    Get-ADDomainController 및 Get-ADDomain cmdlet을 통해 도메인 및 포리스트
    관리가 지원됩니다.

  Active Directory 모듈 Cmdlet 나열
    모든 Active Directory 모듈 cmdlet의 목록을 가져오려면 get-command -module
    ActiveDirectory를 참조하십시오.


  시작
    다음 바로 가기를 클릭하여 Windows PowerShell용 Active Directory 모듈을 쉽게
    시작할 수 있습니다.

       시작 -> 관리 도구 ->
       Windows PowerShell용 Active Directory 모듈

    이 바로 가기를 클릭하면 Active Directory 모듈이 미리 로드된 상태로 Windows PowerShell
    이 열립니다. 또한 Windows PowerShell 프롬프트에서 다음 명령을 입력하여 Active
    Directory 모듈을 가져올 수 있습니다.

      import-module ActiveDirectory
