THEMA
    Übersicht über das Active Directory-Modul

KURZBESCHREIBUNG
    Das Active Directory-Modul ist eine Befehlszeilenschnittstelle zum
    Verwalten von Active Directory.

DETAILBESCHREIBUNG
    Das Active Directory-Modul für Windows PowerShell wurde für IT
    -Experten entwickelt, die Schnittstellenverbindungen mit Active
    Directory herstellen und Active Directory verwalten. Das Active
    Directory-Modul ermöglicht das effiziente Ausführen
    vieler Verwaltungs-, Konfigurations- und Diagnoseaufgaben für
    die Instanzen von Active Directory-Domänendiensten (AD DS) und
    Active Directory Lightweight Directory Services (AD LDS) in den
    betreffenden Umgebungen. Das Active Directory-Modul enthält eine
    Reihe von Windows PowerShell-Cmdlets sowie einen Anbieter. Der
    Anbieter macht die Active Directory-Datenbank über ein
    hierarchisches Navigationssystem verfügbar. Dieses ähnelt dem
    Dateisystem. Wie bei Laufwerken in einem Dateisystem
    (beispielsweise "C:") können Sie Windows PowerShell-Laufwerke
    mit Active Directory-Domänen und AD LDS sowie mit Active 
    Directory-Snapshots verbinden.
    

  Codeabdeckung der Cmdlets des Active Directory-Moduls
    Erstellungs-, Lese-, Aktualisierungs- und Löschvorgänge für Active
    Directory-Objekte werden von Cmdlets wie "New-ADUser", 
    "Get-ADOrganizationalUnit", "Set-ADComputer" und "Remove-ADUser" 
    unterstützt.

    Die Konto- und Kennwortrichtlinienverwaltung wird von Cmdlets wie
    "Enable-ADAccount", "Unlock-ADAccount", "New-ADServiceAccount",
    "Set-ADAccountControl" und "Remove-ADFineGrainedPasswordPolicy" 
    unterstützt.
    
    Die Domänen- und Gesamtstrukturverwaltung wird von Cmdlets
    wie "Get-ADForest", "Set-ADForest", "Set-ADForestMode", 
    "Enable-ADOptionalFeature", "Get-ADDomainController" und "Get-ADDomain" 
    unterstützt.


  Aufführen der Cmdlets des Active Directory-Moduls
    Wenn Sie eine Liste aller Cmdlets für das Active Directory-Modul
    abrufen möchten, geben Sie den Befehl"get-command -module ActiveDirectory"
    ein.
    

  Erste Schritte
    Um das Active Directory-Modul für Windows PowerShell zu starten,
    müssen sie lediglich auf die folgende Verknüpfung klicken:

       Start -> Verwaltung ->
       Active Directory-Modul für Windows PowerShell

    Dadurch wird Windows PowerShell geöffnet, wobei das Active Directory-Modul
    bereits geladen ist. Darüber hinaus können Sie zum Importieren des Active 
    Directory-Moduls in jeder Windows PowerShell-Eingabeaufforderung den 
    folgenden Befehl eingeben:

      import-module ActiveDirectory
