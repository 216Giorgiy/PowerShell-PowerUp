РАЗДЕЛ
    Общие сведения о модуле Active Directory

КРАТКОЕ ОПИСАНИЕ
    Модуль Active Directory представляет собой интерфейс командной
    строки для управления Active Directory.

ПОЛНОЕ ОПИСАНИЕ
    Модуль Active Directory для Windows PowerShell предназначен для
    ИТ-специалистов, которые администрируют и работают с Active
    Directory. Модуль Active Directory позволяет эффективным образом
    выполнять различные задачи администрирования, настройки и диагностики
    в экземплярах доменных служб Active Directory (AD DS) и служб
    Active Directory облегченного доступа к каталогам (AD LDS) в
    соответствующих средах. В модуль Active Directory входит набор
    командлетов Windows PowerShell и поставщик. Поставщик позволяет
    получать доступ к базе данных Active Directory с помощью
    иерархической навигационной системы, напоминающей файловую
    систему. Как и при работе с дисками файловой системы, например
    с диском C:, можно подключать диски Windows PowerShell к
    различным доменам Active Directory и службам AD LDS, а также к
    снимкам Active Directory.

  Описание командлетов, входящих в модуль Active Directory
    Действия Create, Read, Update и Delete поддерживаются для объектов
    Active Directory с помощью таких командлетов, как New-ADUser,
    Get-ADOrganizationalUnit, Set-ADComputer и Remove-ADUser.

    Управление учетными записями и политиками паролей обеспечивается
    такими командлетами, как Enable-ADAccount, Unlock-ADAccount,
    New-ADServiceAccount, Set-ADAccountControl и
    Remove-ADFineGrainedPasswordPolicy.
    
    Управление доменами и лесами обеспечивается такими командлетами,
    как Get-ADForest, Set-ADForest, Set-ADForestMode, Enable-ADOptionalFeature,
    Get-ADDomainController и Get-ADDomain.


  Список командлетов, входящих в модуль Active Directory
    Чтобы получить список всех командлетов модуля Active Directory, воспользуйтесь
    командой get-command -module ActiveDirectory.

  Начало работы
    Чтобы начать работу с модулем Active Directory для Windows
    PowerShell, достаточно просто щелкнуть следующий ярлык:

       Пуск -> Администрирование ->
       Модуль Active Directory для Windows PowerShell

    Этот ярлык позволяет запустить Windows PowerShell с
    предварительной загрузкой модуля Active Directory. Кроме того,
    для импорта модуля Active Directory можно ввести следующую
    команду в любую командную строку Windows PowerShell:

      import-module ActiveDirectory
